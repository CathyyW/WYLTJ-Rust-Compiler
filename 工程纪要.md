# 工程纪要

日期：2026-06-09

## 需求基线

需求来源为根目录 `【Rust版】大作业2：中间代码生成器.pptx`。本阶段按 PPT 全量规则推进，不只满足第 25 页的最低要求。

已拆解任务：

1. 0.1、0.2、0.3：变量属性、类型、左值和可取引用基础。
2. 1.1 至 1.5：函数、语句、return、形参、返回类型。
3. 2.0 至 2.3：变量声明、赋值、声明初始化、类型推断、重影。
4. 3.1 至 3.5：表达式、比较、加减、乘除、函数调用。
5. 4.1 至 4.3：if、else、else if。
6. 5.0 至 5.4：while、for、loop、break、continue。
7. 6.1 至 6.4：不可变变量、不可变引用、可变引用、解引用赋值。
8. 7.0 至 7.4：函数表达式块、块作为表达式、if 表达式、loop 表达式。
9. 8.1 至 8.3：数组类型、数组表达式、数组元素访问和赋值。
10. 9.1 至 9.3：元组类型、元组表达式、元组元素访问和赋值。

## 实现记录

### 编译流水线入口

核心实现：

- `src/app/compiler/analyzer.ts:199-285`：`analyzeSource` 串联词法、语法、语义分析和中间代码生成。词法或语法错误会提前返回；语义通过后生成四元式。

### 四元式抽样

样例：

```rust
fn f(mut x:i32, mut y:i32)->i32 {
  let mut t=x*x+x;
  if t>0 {
    t=t+x*y;
  } else {
    t=t-1;
  }
  t
}
```

结果：通过，生成函数入口、参数、乘法、加法、条件跳转、else 标签、赋值和 return 四元式。
